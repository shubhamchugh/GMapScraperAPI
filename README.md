# GMapScraperAPI

First Command

```sh
npm install
```


Second Command

```sh
npm i puppeteer
```



Third Command
```sh
npm install pm2 -g
```

Start Nodejs Server
```sh
pm2 start MyApiApp.js -i max
```


For Monitor NodeJs Server
```sh 
pm2 monit
```
for check List all processes:
```sh
pm2 list
```

action
```sh
pm2 stop
pm2 restart
pm2 delete
```
